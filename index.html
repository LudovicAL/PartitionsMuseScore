Trad music incipits generator

About

This web page lets you render incipits from a database of traditional music, specifically from the Irish and Quebec repertoires. Incipits are the first (and optionally last) few notes of a tune.

When you choose a tune, it is added to a given "set" on the left portion of the page. You can add sets, remove them or move them up and down. You can remove tunes from a set, or add the same tune multiple times.

Two panels are displayed on the right side of the page, one on top of the other. The first panel contains the ABC code generated from the file you have selected. You can modify that ABC code. The second panel contains the rendering of the ABC code. You can print that rendering.

This tool is provided in conjunction with other existing tools from the same author. Please visit https://ludovical.github.io/Partitions/ to access the main project.
